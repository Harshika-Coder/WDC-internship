# WDC Internship - Month 1 Assignments

Student management and reporting examples built with Node.js.

## Overview

Small collection of scripts demonstrating basic JavaScript/Node.js skills:

- `Month-1/Week-1/Student.js` — sample student data and utility functions
- `Month-1/Week-1/report.js` — reads `students.json` and generates a report
- `Month-1/Week-1/students.json` — sample data file

## Features

- Calculate average marks
- Filter students by marks and city
- Assign grades based on configurable passing marks

## Prerequisites

- Node.js (v16+ recommended)

## Setup

1. Install dependencies (if any):

```bash
npm install
```

2. Create a `.env` file in the project root with the following content (optional):

```
PASSING_MARKS=40
```

## Run

From the project root run:

```bash
node Month-1/Week-1/report.js
```

Or run the Student example directly:

```bash
node Month-1/Week-1/Student.js
```

## Project structure

- `Month-1/Week-1/` — Week 1 exercises
  - `Student.js` — example script (logging students, filters, averages)
  - `report.js` — report generator that reads `students.json`
  - `students.json` — sample data
- `README.md` — this file
